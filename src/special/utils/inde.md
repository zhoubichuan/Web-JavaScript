# 代码规范
- 不要一行声明多个变量
- 请使用 ===/!==来比较 true/false 或者数值
- 使用对象字面量替代 new Array 这种形式
- 不要使用全局函数
- Swtich 语句必须带有 default 分支
- If 语句必须使用大括号
- for-in 循环中的变量 应该使用 var 关键字明确限定作用域， 从而避免作用域污染
